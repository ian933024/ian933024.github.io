# 歷屆教檢考古題內容分析

這個資料夾是一個可部署到 GitHub Pages 的純靜態網頁。

## 使用方式

將整個資料夾內容放到 GitHub Pages 對應的公開目錄後，開啟 `index.html` 即可使用。

若要在本機預覽，請用靜態伺服器開啟，例如：

```bash
python3 -m http.server 8765
```

再到：

```text
http://127.0.0.1:8765/
```

不要直接用 `file://` 開啟，因為瀏覽器可能會阻擋 JSON 資料載入。

## 資料內容

- `data/questions.json`：逐題分析資料
- `data/frequency.json`：本題重點頻率分析
- `data/meta.json`：資料來源與整理方法
- `assets/本題重點文字雲.png`：前一步產出的文字雲圖片
- `index.html`：互動頁面入口
- `styles.css`：頁面樣式
- `app2.js`：簡化版選取查詢與統計功能

## 資料來源

- `考古題/題目`：110-114 年國民小學教師資格考試題目 PDF
- `考古題/答案`：110-114 年國民小學教師資格考試選擇題參考答案 PDF
- `陳嘉陽 精簡1.pdf`、`陳嘉陽 精簡2.pdf`：分類判斷參考筆記

## 整理方法

1. 使用 PDF 文字擷取工具將考古題依題型與題號切成逐題資料。
2. 逐題標註「小科目」與「本題重點」。
3. 以「本題重點」為單位進行頻率統計。
4. 將資料轉為靜態 JSON，頁面在瀏覽器端即時篩選與統計。
