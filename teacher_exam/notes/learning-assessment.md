# 教檢複習筆記：學習評量 × 評量觀、試題分析與多元評量

> 涵蓋：三種評量觀（of/for/as learning）、形成性與總結性評量、常模參照與標準參照、多元評量類型（實作、動態、適性、真實、檔案）、評分規準、信度與效度、試題分析（難度、鑑別度）、標準分數

---

## 一、評量的目的與觀點

學習評量依目的可以分為三種，近年來常以英文術語直接入題：

- **對學習的評量（Assessment OF Learning）**：在學習結束後評定學習成果，給出一個判斷性的結果，是傳統意義上的總結性評量，例如期末考、段考後的成績通知。目的是「評定」，不以改變教學為主要考量。
- **為了學習的評量（Assessment FOR Learning）**：在教學過程中進行，目的是提供教師和學生即時回饋，作為調整教學與學習的依據。對應的是**形成性評量**，包含隨堂小考、作業、課堂問答等。
- **評量即學習（Assessment AS Learning）**：評量本身就是學習活動。當學生透過評量反思自己「哪裡懂、哪裡不懂」並主動調整，評量就成為培養**後設認知**與**自我調整能力**的手段。具體的實踐策略包括：讓學生在寫報告前先設定評量標準、引導學生進行自我評量或同儕互評、要求學生針對評量結果寫下反思等。

三種評量觀的核心差異是「**誰從評量中獲益**」：OF 是給外部利害關係人（家長、學校）一個結果；FOR 是給教師調整教學的資訊；AS 是讓學生自己成為評量過程的主體。

形成性評量和總結性評量是另一個常見考點。**期末考是總結性評量**，不算形成性評量，這是最常被誤選的地方。診斷性評量通常在教學前進行，目的是了解學生的起點行為與先備知識。

> **代表考題【113年選8】** 下列何者可以進行形成性評量？甲隨堂小考、乙作業、丙課堂問答、丁期末考 → **(A) 甲乙丙**。期末考是總結性評量，不屬於形成性評量。
>
> **代表考題【113年選25】** 郭老師依據評分規準對學生製作的導覽影片進行評量，最符合哪種取向？→ **(C) 對學習的評量（Assessment OF Learning）**。老師拿既定評分規準評定最終成果，是總結性的評定。

## 二、常模參照 vs 標準參照（效標參照）

這是近年每屆幾乎必考的概念，判斷時只需抓住一個核心問題：**這個評量結果是跟別人比，還是跟標準比？**

**常模參照**的評量結果是相對的，必須放在群體脈絡中才有意義。PR值、百分等級、「前20%入選複賽」、「大學學科能力測驗四級分」都是常模參照，因為分數或名次的意義是由其他考生的表現決定的。

**標準參照（效標參照）**的通過條件是固定的、與他人無關的。只要達到既定表現水準就算通過，不管其他人考幾分。機車駕照、英語檢定（全民英檢）、以公告的表現水準（如1到6級）確認學習成就等級，都是標準參照。

最容易混淆的選項是**研究所入學考試**和**縣市教師甄試**：看起來有一個「通過」的門檻，但實際上錄取與否是由競爭排名決定，今年通過、明年同樣成績可能不通過，因此本質上仍是**常模參照**。

| | 常模參照 | 標準參照（效標參照）|
|--|---------|-------------------|
| 比較基準 | 與他人的相對位置 | 與固定表現標準的距離 |
| 典型例子 | 班排名、PR值、學測級分、甄試入選 | 駕照考試、英語檢定、表現水準分級 |
| 分數意義 | 相對於群體 | 絕對達成程度 |

> **代表考題【113年選2】** 最符合標準參照評量的是？→ **(B) 通過機車駕照考試**。達到固定標準就通過，與他人無關。(C)(D) 研究所入學、教師甄試雖有標準，但本質是競爭選拔，屬常模參照。

## 三、多元評量的類型

多元評量是相對於傳統紙筆測驗的一組評量方式，重視真實情境、學習歷程與多樣的能力展現，但「多元」並不代表排除紙筆測驗，而是依目標選擇最適合的方式。

### 主要類型的辨別

幾個容易混淆的類型需要記清楚定義：

**真實評量（Authentic Assessment）** 強調評量情境盡可能貼近真實生活，讓學生在接近現實的任務中展現能力，而非在人工化的考試情境中作答。

**實作評量（Performance Assessment）** 讓學生實際執行某項任務或展示技能，師資生在實習中進行教學演示就是實作評量。真實評量強調「情境擬真」，實作評量強調「實際操作展示」，兩者有重疊但不相同。

**動態評量（Dynamic Assessment）** 以**維高斯基（Vygotsky）**的最近發展區（ZPD）為理論基礎，在評量過程中給予提示或協助，藉此評估學生的**學習潛力**，而非僅測量現有能力。注意：動態評量的基礎是維高斯基，**不是皮亞傑**。

**適性評量** 依學生的個別差異調整題目難度，讓不同能力的學生接受適合自己程度的測驗。適性評量強調「因材施測」，**不等於**真實評量的「情境擬真」。

> **代表考題【112年選14】** 下列有關學習評量的敘述，何者正確？→ **(B) 師資生在教育實習中進行教學演示，屬於實作評量**。(A) 適性評量≠真實評量；(C) 動態評量的理論基礎是維高斯基不是皮亞傑；(D) 教師資格考試是標準化測驗，不是真實評量。

### 檔案評量（Portfolio Assessment）

檔案評量透過收集學生一段時間內的作品或學習記錄，呈現**學習歷程**而非單一時間點的結果，特別重視學生的自我省思。幾個關於檔案評量的常見錯誤觀念：

- 檔案資料**需要篩選**，不是越多越好。有意識地挑選有代表性的作品才能呈現學習軌跡。
- 結構式與非結構式的呈現方式**各有適用情境**，並非結構式一定更好。
- 檔案評量重視**質性意義的呈現**，而非分數的量化比較。

> **代表考題【110年選24】** 哪些符合檔案評量的敘述？甲透過檔案做學習歷程的回顧與省思、乙檔案內資料應盡可能完整不須篩選、丙結構式比非結構式更好、丁重視質性意義而非分數比較 → **(B) 甲丁**。乙和丙的說法都錯誤。

### 評分規準（Rubric）

評分規準是事先制定並公開的評分標準，列出不同面向的表現水準，讓評量更透明、一致。它既是評量工具，也是教學工具——讓學生事先知道評分標準，本身就有引導學習的功能。

評分規準可用於教師評量、學生**自我評量**和同儕互評，各有優缺點：

| 評量方式 | 優點 | 可能限制 |
|----------|------|----------|
| **教師評量** | 標準一致，減少主觀性；對所有學生公平 | 耗時；可能過於制式，忽略創意或個別表現 |
| **自我評量** | 培養後設認知，學生學會自我監控與調整 | 學生可能評分不客觀（高估或低估自己）|

問答題常問「除了教師評量和自我評量外，還能使用何種方式？」標準答案是**同儕互評**——學生互相評量，能從他人的作品中學習，也能以評量者的角度深化對評分標準的理解。

預防**月暈效應**（被第一印象影響後續評分）的方法是**按題批改**（先批所有人的第一題，再批第二題）以及**事先制定評分規準**。「批改完一位學生的全部試卷再換下一位」是按人批改，容易產生月暈效應。

> **代表考題【110年選21】** 最能避免月暈效應的兩種方式？→ **(C) 乙丙**（閱卷前擬訂評分規準 ＋ 請兩位老師分別評分）。「批改完一位學生後再換下一位」（丁）是按人批改，反而容易有月暈效應。

## 四、信度與效度

**效度（Validity）** 是測驗最重要的品質指標，指的是「測驗是否真的測到了它想測的東西」。一份測驗可以很穩定（高信度），但測的東西不對（低效度），所以效度比信度更根本。提升效度的方法包含：確保試題能代表課程目標（**內容效度**）、根據教學目標命題（試題與教學目標對齊）。

**信度（Reliability）** 是測驗結果的一致性與穩定性——同一份試卷在相似條件下施測，應該得到類似的結果。影響信度的主要因素：

- **題數**：題目越多，信度越高。30題比20題更能穩定反映能力。
- **題型**：填充題沒有猜題因素，隨機誤差低，信度優於選擇題。選擇題雖然評分客觀，但猜對的機率（猜測因素）會引入隨機誤差，反而降低信度。主觀評分的申論題或作文因評分者差異大，信度最低。
- **評分一致性**：評分規準越清楚、評分者越多，信度越高。

因此在題數和題型都不同的情況下，**30題填充題的信度最高**（題數最多且無猜題因素）。

> **代表考題【111年選16】** 考試範圍與題目品質相同，哪個測驗信度較高？選項：20題選擇題 / 20題填充題 / 30題選擇題 / 30題填充題 → **(D) 30題填充題**。題數多（比20題信度高）且無猜題因素（比選擇題信度高），兩個提升信度的因素都佔到了。

提升作文比賽（主觀評量）信效度的具體作法：
- **提升效度**：制定明確的評分向度，確保評量內容能代表「作文能力」的核心要素（例如：內容、組織、語法、創意各佔比重）。
- **提升信度**：請多位評分者獨立評分後再取平均，或採用評分規準讓評分標準一致，也可採按題批改方式降低月暈效應。

## 五、試題分析：難度、鑑別度與標準分數

試題分析是評量後檢視試題品質的重要程序，核心指標是**難度（P）**和**鑑別度（D）**，考題常出現計算題或解釋題。

### 難度（P）

難度指的是全體受試者的**答對率**，計算方式為：

**P = ( 高分組答對率 + 低分組答對率 ) ÷ 2**

P 值介於 0 到 1 之間，**數值越高代表題目越容易**。理想的難度約在 **0.5** 左右，此時高低分組的差異最大，鑑別效果最好。P 值過高（太簡單）或過低（太難）都不利於鑑別學生能力。

### 鑑別度（D）

鑑別度衡量一道題目是否能區分高分組和低分組：

**D = 高分組答對率 − 低分組答對率**

D 值介於 -1 到 1，**越高越好**。一般而言 D ≥ 0.3 才算具有一定鑑別力；D < 0.2 表示鑑別力不足，應重新檢討該題。若 D 為負值，代表高分組答對率反而低於低分組，表示該題有嚴重問題（可能題意不清或正確答案有爭議）。

計算範例（114年考題）：班上30人，高低分組各8人，第一題高分組答對人數比低分組多3人：

D = 3 ÷ 8 = **0.375**

另一個常見計算情境是從選答率表格讀取數據：若正確選項（＊）高分組選答率為 0.86、低分組為 0.50，則 P = (0.86 + 0.50) / 2 = 0.68，D = 0.86 - 0.50 = 0.36。

### 誘答力

除了正確選項，干擾選項的誘答力也是試題品質的重要指標。**理想的誘答選項應該對低分組有吸引力，但對高分組沒有吸引力**。若高低分組在各選項的分布非常接近（接近隨機），表示學生都在猜測，可能是題意不清或該概念尚未確實教過。

> **代表考題【114年選24】** 高低分組各8人，高分組比低分組多答對3人，鑑別度為何？→ **(C) 0.375**（3 ÷ 8 = 0.375）。
>
> **代表考題【110年選17】** 試題品質指標最佳的題目？高分組/低分組答對率：第1題 80%/20%、第2題 20%/50%、第3題 65%/55%、第4題 50%/10% → **(A) 第1題**。D = 0.6，鑑別度最高；P = 0.5，難度適中。第2題鑑別度為負值（題目有問題），第3題幾乎沒有鑑別力。

### 標準分數（Z 分數）

當需要比較學生在不同科目（平均數和標準差各異）的相對表現時，原始分數不能直接比較，必須轉換成**標準分數（Z 分數）**：

**Z = ( 個人分數 − 平均數 ) ÷ 標準差**

Z 值越大，代表相對於該科目的表現越好。

計算範例（111年）：麗雪在各科的 Z 值：
- 國語：(84 - 88) / 8 = **-0.5**
- 數學：(80 - 71) / 6 = **+1.5**
- 自然：(74 - 84) / 10 = **-1.0**
- 社會：(86 - 80) / 12 = **+0.5**

數學的 Z 值最高（+1.5），代表麗雪相對於同學的表現，在數學科最為突出——即使原始分數只有80分，低於社會科的86分，但考慮全體的平均和分散程度後，數學才是她表現最好的科目。

## 易混淆考點

- **三種評量觀**：OF = 結果評定（期末考）；FOR = 形成性回饋（隨堂小考、作業）；AS = 評量本身即是學習活動（自我評量、反思）。
- **期末考是總結性評量**，不是形成性評量，不論題目改得多靈活都一樣。
- **動態評量 = Vygotsky（ZPD）**，考試常把基礎說成皮亞傑當陷阱。
- **適性評量 ≠ 真實評量**：適性是「因材施測」；真實是「情境擬真」。
- **研究所入學、教師甄試**看似有標準，實為競爭排名，是**常模參照**，不是標準參照。
- **鑑別度**公式 = 高分組答對率 **減** 低分組答對率；負值表示題目有問題。
- **難度 P 越高越容易**（P = 答對率），理想值約 0.5。
- **Z 值**才能跨科目比較，原始分數高不代表相對表現好。
- **按題批改**（不是按人批改）才能有效預防月暈效應。
- **檔案評量**需要篩選資料，重視質性呈現，不以分數比較為主要目的。
