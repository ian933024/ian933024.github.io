# 教檢複習筆記：教育哲學五大流派

> 涵蓋：永恆主義、實用主義、建構主義、存在主義、後現代主義

---

## 一、永恆主義（Perennialism）

### 理論背景與核心主張

永恆主義源自古希臘哲學，尤其是柏拉圖和亞里斯多德的傳統，強調宇宙間存在著**客觀、普遍、永恆不變的真理**，不因時代、文化或個人而改變。人類理性是認識真理的最高工具，教育的核心任務就是訓練理性、引導學生趨近這些永恆的觀念。

二十世紀由哈欽斯（Robert Hutchins）與阿德勒（Mortimer Adler）在美國系統化，發展出以「古典名著計畫」（Great Books Program）為核心的課程模式，認為西方文明中累積的經典著作——如柏拉圖《理想國》、亞里斯多德《尼各馬可倫理學》、聖奧古斯丁《懺悔錄》等——蘊含了跨時代的人類智慧，是所有學生都應深入研讀的核心教材。馬里旦（Jacques Maritain）則從天主教哲學角度出發，強調教育目的在使人認識永恆的神聖秩序。

### 知識觀與真理觀

永恆主義的知識是**先驗的（a priori）**：真正的知識不依賴感官經驗，而是透過理性推理所得。這與實用主義的「知識是工具」截然不同——永恆主義視知識為目的本身，而非達到其他目的的手段。真理一旦確立，就不因時代更迭而改變：數學定理、邏輯法則、形而上的道德原則皆屬之。

### 課程與教學

| 面向 | 永恆主義主張 |
|------|-------------|
| 核心教材 | 古典名著（Great Books）：文學、哲學、神學、數學、科學經典 |
| 課程目的 | 培養能理性思考、追求永恆真理的人格 |
| 教師角色 | 知識的傳遞者與理性的引導者，具有相對的知識權威 |
| 教學方法 | 蘇格拉底式對話（產婆法）、講述、討論——目的是啟發學生內在的理性能力 |
| 學生角色 | 被動接受，但需主動透過理性思考加以內化 |

**蘇格拉底產婆法**的精髓在於：教師透過連續追問，幫助學生從自身內部「生出」已潛藏的理性知識，而非從外部灌輸。這與永恆主義「人人皆有理性，可藉此趨近真理」的信念一致。

### 代表人物速記

- **哈欽斯（Hutchins）**：芝加哥大學校長，力推 Great Books 課程
- **阿德勒（Adler）**：Paideia 計畫——強調共同核心課程，人人接受相同品質的教育
- **馬里旦（Maritain）**：天主教永恆主義，強調靈魂與理性的整合

## 二、實用主義（Pragmatism）

### 理論背景與核心主張

實用主義是十九世紀末至二十世紀初在美國興起的本土哲學，主要代表人物包括皮爾士（C. S. Peirce）、詹姆士（W. James）與杜威（J. Dewey）。其核心命題是：**一個觀念的意義與真假，取決於它在實際行動中產生的效果**——有用即真（truth is what works）。

杜威是實用主義在教育上最重要的思想家。他反對將學校與社會割裂，主張**教育即生長（Education as growth）、教育即生活（Education as life）、教育即民主（Education as democracy）**。在他看來，兒童的經驗是課程的起點，知識的獲得必須透過主動解決問題、與環境互動，而非被動記憶教師灌輸的內容。

### 知識觀

實用主義的知識是**工具性的、後驗的、情境性的**：知識來自於人與環境的互動和問題解決過程，沒有絕對的、超越時空的真理。知識必須在實際情境中接受「效用」的檢驗，無效的知識就失去了其意義。這是對永恆主義「先驗真理」的直接挑戰。

### 課程與教學

| 面向 | 實用主義主張 |
|------|-------------|
| 課程核心 | 學生的生活經驗、當下的社會問題 |
| 教師角色 | 引導者、促進者（非知識權威） |
| 教學方法 | 問題解決、探究學習、做中學（learning by doing） |
| 評量 | 過程導向，觀察學生如何解決問題，而非測驗知識記憶 |
| 學生角色 | 主動的探究者與問題解決者 |

**克伯屈（W. H. Kilpatrick）的設計教學法（Project Method）**是實用主義在課程上最著名的應用，強調以「問題」為中心，四個步驟：
1. **確定目的**（學生提出真實的目標或問題）
2. **擬訂計畫**（師生共同討論解決方案）
3. **實行計畫**（執行、嘗試、行動）
4. **評量結果**（反思、修正、再出發）

這四個步驟與杜威的「反省思維」（reflective thinking）一脈相承：遇到疑難情境→界定問題→提出假設→蒐集資料→驗證假設。

### 杜威與 108 課綱的關聯

108課綱的「核心素養」強調學習與真實生活的連結、知識技能態度的統整、關注個體全面發展——這與杜威「教育即生長、學校即社會」的主張有高度共鳴。兩者都反對將教育目的定位在「為未來職業做準備」的狹義工具觀，而是強調在學習過程中當下即具有意義的成長。

### 代表人物速記

- **杜威（J. Dewey）**：教育即生長、做中學、民主與教育
- **克伯屈（W. H. Kilpatrick）**：設計教學法（問題中心）
- **詹姆士（W. James）**：「有用即真」的工具主義真理觀

## 三、建構主義（Constructivism）

### 理論背景與核心主張

建構主義不是單一的哲學派系，而是一組關於**知識如何被習得**的學習理論，核心命題是：**知識不是被動接受的，而是學習者主動建構的**。學習者根據已有的先備知識與經驗，對新資訊進行詮釋與整合，從而形成個人的理解結構。

建構主義大致可分為兩個主要流派：

#### 認知建構主義（皮亞傑 Jean Piaget）

皮亞傑從發展心理學出發，描述兒童如何透過與物理環境的互動建構知識：
- **同化（Assimilation）**：將新資訊納入既有的認知基模（schema），不改變基模本身。例如：兒童看到貓叫「狗狗」，因為符合他的「四隻腳的動物＝狗狗」基模。
- **調適（Accommodation）**：既有基模無法容納新資訊時，修改或新建基模。例如：發現「貓」和「狗」不一樣，建立新的區分。
- **平衡（Equilibration）**：在同化與調適之間不斷達到新的認知平衡，驅動認知發展。

皮亞傑的四個認知發展階段（感覺動作期、前運思期、具體運思期、形式運思期）說明了兒童在不同年齡段能進行何種程度的抽象思考，對課程設計有直接的啟示：**教材的抽象程度必須符合兒童的認知發展階段**。

#### 社會建構主義（維高斯基 Lev Vygotsky）

維高斯基強調社會互動與語言在知識建構中的核心角色，提出：
- **最近發展區（Zone of Proximal Development, ZPD）**：兒童獨立能完成的任務水準，與在他人協助下能完成的水準之間的差距。教學應瞄準 ZPD，提供適當的挑戰。
- **鷹架（Scaffolding）**：教師或同儕在學習者的 ZPD 內提供的暫時支持，隨學習者能力提升而逐步撤除。
- **語言的中介功能**：語言不只是溝通工具，也是思考的工具。兒童透過「私語」（private speech）自我調節行為與思考，逐漸內化為內在語言。

皮亞傑 vs 維高斯基的主要差異：皮亞傑認為認知發展引領語言發展，維高斯基認為語言（社會互動）引領認知發展。

### 課程與教學含義

| 面向 | 建構主義主張 |
|------|-------------|
| 知識觀 | 知識是個人（或社會）建構的，無絕對客觀的知識 |
| 教師角色 | 引導者、鷹架提供者，而非知識傳輸者 |
| 教學設計 | 提供真實情境問題，開放探究空間 |
| 學習活動 | 合作學習、同儕討論、小組探究 |
| 評量 | 重視學習歷程、概念圖、作品集（portfolio） |

建構主義衍生的教學模式包括：
- **問題導向學習（Problem-Based Learning, PBL）**：以真實且複雜的問題為起點
- **錨式學習（Anchored Instruction）**：以豐富的情境故事為「錨」，引導知識應用
- **認知學徒制（Cognitive Apprenticeship）**：師傅示範，學徒在真實情境中練習

### 建構主義 vs 傳統教學

| | 傳統教學（行為主義） | 建構主義教學 |
|---|---|---|
| 知識來源 | 教師傳遞，學生接受 | 學生主動建構 |
| 教師角色 | 知識的傳輸者 | 引導者、提問者 |
| 學習活動 | 反覆練習、背誦 | 探究、討論、解決真實問題 |
| 評量 | 標準化測驗 | 歷程評量、真實評量 |

### 代表考題

**【113年選7】** 林老師採用建構主義進行教學，何者最能展現其教學理念？
→ **(C) 提供開放的主題，鼓勵學生自行選擇研究方向進行小組探究**。
建構主義強調學習者的主動性與選擇空間，以及社會互動（小組）。反例：(A)提供學生「須了解的內容」是傳統教師中心；(B)要求背誦是行為主義；(D)讓所有學生按相同步驟是標準化教學，違反建構主義對個體差異的尊重。

## 四、存在主義（Existentialism）

### 理論背景與核心主張

存在主義是二十世紀興起於歐洲的哲學運動，以人的「存在」與「選擇」為核心問題，反對任何試圖從外部（宗教、科學、社會）為人的生命預先定義「本質」的嘗試。

最著名的命題來自沙特（Jean-Paul Sartre）：**「存在先於本質」（Existence precedes essence）**。傳統哲學（如宗教、理性主義）認為人有一個預先設定的本質或目的（上帝設計的、理性規定的），而沙特認為：人先存在於世界，再透過自己的選擇與行動，創造自己的本質。換言之，**人是絕對自由的，同時也必須為自己的選擇承擔全部責任**。沙特稱此為「存在的焦慮」（anguish）——自由本身是沉重的負擔。

海德格（Martin Heidegger）則強調「此在（Dasein）」——人作為一種「在世界中存在」的存有，必須面對死亡的有限性，從而激發出對「本真生活（authenticity）」的追求：不隨波逐流，而是真正地活出自己的存在。

馬丁·布伯（Martin Buber）提出的**「我-你（I-Thou）」與「我-它（I-It）」關係**對教育影響深遠：
- **我-它關係**：將他人（或對象）視為工具或客體加以利用，是功能性的關係
- **我-你關係**：真正的相遇（encounter），視對方為完整的主體，產生真實的對話與連結

布伯認為真正的教育應建立在「我-你」的師生關係上，而非教師把知識「傳輸」給學生（我-它）。

### 存在主義的教育主張

| 面向 | 存在主義主張 |
|------|-------------|
| 教育目的 | 協助學生認識自身的存在處境、做出真實的選擇並承擔責任 |
| 教師角色 | 促進者（facilitator）——不提供答案，而是幫助學生面對問題、發現意義 |
| 師生關係 | 布伯的「我-你」關係：真誠的相遇與對話 |
| 課程 | 關注個人意義與生命問題，反對標準化課程 |
| 教學方法 | 開放討論、個人反思、真實情境的生命議題探討 |
| 反對 | 外部強加的「正確答案」、以訓練或效率為導向的教育 |

存在主義強調**個人的獨特性與主體性**，因此反對把學生視為可被同質化管理的對象。每個學生的存在處境不同，教師的任務是陪伴學生面對困境、思考意義，而非提供標準解答。

### 存在主義 vs 精粹主義

| | 存在主義 | 精粹主義（Essentialism） |
|---|---|---|
| 知識觀 | 個人意義的建構，無客觀普遍知識 | 有客觀、基本的核心知識必須傳授 |
| 課程 | 以個人生命問題為中心 | 以系統性學科知識為中心 |
| 教師 | 促進者，不提供答案 | 知識傳遞者，引導學生精熟 |
| 學習目標 | 自由選擇、承擔責任、活出本真 | 精熟基本知能，為社會做準備 |

精粹主義（Essentialism）認為有一套「精粹的」基本知識學生必須精熟，教師應有效地傳遞這些知識——這與存在主義的「個人意義建構」截然對立。

### 代表考題

**【113年選2】** 下列哪一項與沙特「存在先於本質」最相符？
→ **(C) 成為你自己選擇成為的樣子**。「存在先於本質」意謂：沒有預設的人生腳本，你透過選擇與行動創造自己的樣子。(A)(D)都是外部期待定義你，(B)是命定論。

**【110年綜5】** 李老師以存在主義作為教學方案基礎，何者最符合其教學目標？
→ **(C) 教師是促進者，討論並無正確答案，而是幫助同學了解如何抉擇、如何設定目標、並能為自己的選擇負責**。注意排除(A)(B)（傳遞知識/訓練心智→精粹或永恆主義）和(D)（傳遞社會價值→重建主義）。

**【111年選8】** 李老師認為「把最重要、最基本的知識教給學生，不必要浪費時間從嘗試錯誤中學習」，較接近哪種教育哲學？
→ **(A) 精粹主義**。強調有效傳遞「基本知識」，反對摸索──這是精粹主義（非存在主義）的立場，本題作為對照。

## 五、後現代主義（Postmodernism）

### 理論背景與核心主張

後現代主義是二十世紀後半葉興起的思想浪潮，對現代性（modernity）的基本假設提出全面的質疑。現代性相信：存在客觀真理、理性能揭示普遍規律、科學可帶來進步、宏大敘事（如民主、自由、理性）能為全人類指引方向。後現代主義對這一切都抱持懷疑。

李歐塔（Jean-François Lyotard）的核心論述：**「對宏大敘事（grand narratives）的不信任」**。任何宣稱自己代表普遍真理、能詮釋全人類處境的宏大框架（基督教、馬克思主義、啟蒙理性）都是一種壓制差異的權力運作。

傅科（Michel Foucault）揭示**權力與知識的不可分割性**：什麼被視為「知識」、什麼被視為「真理」，都是特定歷史時期的權力運作結果。學校、醫院、監獄等現代機構都是「規訓（discipline）」的場所，透過全景敞視（panopticon）般的監視機制，形塑服從的主體。課程知識絕非中立，而是充滿意識形態的選擇與排除。

德希達（Jacques Derrida）的**解構（deconstruction）**：語言中的意義從不固定，總是在「延異（différance）」中不斷滑動，沒有「純粹的」原始意義可以被掌握。「作者已死」（羅蘭·巴特 Roland Barthes）：文本的意義不由作者決定，而是在讀者的閱讀行為中被生產——每次閱讀都是一次新的意義建構。

### 後現代主義的四「反」

| 立場 | 說明 |
|------|------|
| **反本質主義**（Anti-essentialism） | 不存在固定不變的「本質」，一切事物的特性都是歷史建構的 |
| **反基礎主義**（Anti-foundationalism） | 不存在可以作為一切知識基礎的終極真理或方法論 |
| **反威權主義**（Anti-authoritarianism） | 反對任何形式的知識或文化霸權，質疑專家、機構的絕對權威 |
| **擁抱多元主義**（Pro-pluralism） | 尊重差異、接受多元詮釋，而非尋求同質化的共識 |

> **考試陷阱**：「反多元主義」**不是**後現代主義的主張。後現代主義積極擁護多元（pluralism），「反多元主義」恰恰是它所反對的現代性態度。

### 後現代主義的教育主張

**課程觀：萬花筒課程（Kaleidoscopic curriculum）**——如萬花筒般，沒有固定的中心圖案，視角一換便呈現不同的美麗圖案。課程不應有單一的核心真理，而應呈現多元、去中心的知識面貌，讓邊緣群體的聲音也能進入課堂。

| 面向 | 後現代主義教育主張 |
|------|-------------------|
| 課程知識觀 | 知識是社會建構的，充滿意識形態，需加以批判解構 |
| 課程設計 | 多元文化、多元觀點、去中心化；重視被邊緣化的知識 |
| 教師角色 | 批判的促進者，引導學生解構主流論述 |
| 教學方法 | 對話、解構文本、批判思考、「作者已死」式的開放詮釋 |
| 增能賦權（empowerment） | 讓教師、學生擁有更多專業自主與決策參與，打破科層控制 |

**艾波（Michael Apple）的意識形態與課程**：課程知識的選擇反映了社會的主流意識形態（資本主義、父權、白人中心等），特定階級的文化知識被合法化、其他被邊緣化。課程是一種意識形態的複製工具。

**傅科的知識/權力論在教育中的應用**：師資培育機構、教科書審查、課程標準等，都是知識被賦予正當性的機制。哪些知識「值得學習」、哪些教學法是「主流」，都是權力運作的結果，而非自然中立的選擇。

### 後現代主義 vs 現代主義（永恆主義/精粹主義）

| | 現代主義立場 | 後現代主義立場 |
|---|---|---|
| 知識 | 客觀、普遍、可驗證的真理 | 社會建構，充滿權力關係 |
| 課程 | 核心知識、共同文化遺產 | 多元、去中心、解構主流 |
| 意義 | 固定，由作者/文本決定 | 流動，由讀者/脈絡建構 |
| 差異 | 需要克服或同化 | 需要尊重與彰顯 |

### 代表考題

**【110年選6】** 哪一項「不是」後現代主義用於教育的觀點？
→ **(B) 反多元主義**。後現代主義擁抱多元，「反多元」是它批判的對象。其他三項（反本質主義、反基礎主義、反威權主義）都是後現代的核心立場。

**【111年問1】** 說明後現代主義主張的兩項教育目的，及三項對課程內容或教學方法的啟示。

| 教育目的 | 說明 |
|----------|------|
| ① 培養批判意識 | 幫助學生解構主流論述，覺察知識背後的權力與意識形態 |
| ② 尊重差異與多元 | 承認並彰顯不同文化、族群、性別的知識與聲音，而非同質化 |

| 課程/教學啟示 | 說明 |
|--------------|------|
| ① 萬花筒課程觀 | 課程不以單一中心知識為準，引入多元觀點與邊緣聲音 |
| ② 解構教學 | 引導學生分析文本背後的意識形態，如「作者已死」式的開放閱讀 |
| ③ 增能賦權 | 教師與學生均享有更多專業自主與決策空間，打破科層管控 |

**【114年非選2】** 說明文化資本造成階級再製的兩項原因；「文化雜食」為何可能是優勢文化資本？

| 文化資本造成階級再製的原因 | 說明 |
|--------------------------|------|
| ① 學校認可特定文化品味 | 學校課程與評量標準以上層階級的「高雅文化」為準，中下層學生的文化實踐不被認可 |
| ② 文化資本轉換為學業成就 | 接觸高雅文化的學生更容易理解學校語言、符號與期待，在學業上取得優勢，形成代際傳遞 |

「文化雜食」（cultural omnivorousness）之所以是優勢文化資本：上層階級不僅掌握高雅文化，還能靈活穿梭於各種文化形式（流行音樂、街頭文化、多元族群文化），展現廣泛的文化適應力與包容性。這種「跨域能力」在當代多元社會中具有更高的社會流動優勢，形成比單一高雅品味更難以被仿效的新型文化資本，從而強化階級再製。

## 六、綜合比較

### 五大流派對照表

| | 永恆主義 | 實用主義 | 建構主義 | 存在主義 | 後現代主義 |
|---|---|---|---|---|---|
| **真理觀** | 客觀、普遍、永恆 | 工具性、效用性 | 個人/社會建構 | 個人意義建構 | 流動、情境性 |
| **知識來源** | 理性（先驗） | 經驗（後驗） | 主動建構 | 個人選擇與反思 | 權力與論述 |
| **課程核心** | 古典名著 | 生活問題/經驗 | 真實情境探究 | 個人生命問題 | 多元去中心 |
| **教師角色** | 理性引導的權威 | 引導者/協助者 | 鷹架提供者 | 促進者 | 批判的促進者 |
| **代表人物** | Hutchins, Adler | Dewey, Kilpatrick | Piaget, Vygotsky | Sartre, Buber | Foucault, Derrida |

### 常見考試陷阱

| 陷阱題目 | 正確答案 | 常見誤解 |
|----------|----------|----------|
| 蘇格拉底產婆法 | 啟發理性的**對話歷程**（重點在方法） | 誤以為等同於「閱讀古典名著」 |
| 康米紐斯（配合實物圖片） | **經驗主義**（感官知覺） | 誤以為是永恆主義（古典傳統） |
| 多元文化納入課程 | **後現代主義**（萬花筒課程） | 誤以為是永恆主義或觀念論 |
| 後現代主義「反多元主義」 | **錯誤**——後現代擁護多元 | 和「反本質主義」混淆 |
| 「作者已死」 | **後現代主義**（意義流動） | 誤以為意義「客觀存在」需被發現 |
| 「精粹主義」vs「存在主義」 | 精粹＝傳授基本知識；存在＝個人選擇與意義 | 兩者都反對填鴨，但理由截然不同 |
| 克伯屈設計教學法 | **實用主義**（問題中心，四步驟） | 與建構主義的 PBL 混淆 |

### 知識類型速記

| 知識類型 | 說明 | 對應哲學 | 例子 |
|----------|------|----------|------|
| **先驗的（a priori）** | 不依賴經驗，靠理性推導 | 永恆主義、理性主義 | 數學定理、邏輯命題 |
| **後驗的（a posteriori）** | 依賴感官經驗 | 實用主義、經驗主義 | 科學實驗的觀察結果 |
| **超驗的（transcendental）** | 超越感官的存在 | 宗教哲學、形上學 | 上帝、靈魂 |
